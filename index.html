About our

6-month Work Integrated Learning Programme

Hello there! Welcome to Magizh Technologies !!!
The 6-month Web Development Program offers comprehensive training in Python programming, competitive problem-solving, and practical experiences. With a limited batch size of 15 students, the program includes weekly assessments, group projects, and a 2-month internship. Key highlights include personalized support, collaborative exposure,
regular progress tracking, and participation in open hackathons.

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

The curriculum covers HTML, CSS, Bootstrap, JavaScript, Python (beginner to advanced), competitive programming, cloud computing (MS Azure), Vue JS, database designing, data structures, system design, deployment, and prompt engineering. Internship opportunities open after three months, and placements are available for up to one year from enrollment.
Mentors include Mythili M, a Senior Software Engineer, and Gokulakrishnan Muthusamy, a Python Developer and Ex-Academic Affairs Coordinator at IIT Madras BS.
